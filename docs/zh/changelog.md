# 更新日志

## V2.0.1

**提升连接稳定性**：USB 传输中断自动恢复，设备无响应时看门狗自动重启。

---

## V2.0.0

从 ESP32S3-USB-Keyboard-To-BLE 闭源分支，升级为键鼠组合 HID 桥接。

**新增功能：**
- USB 鼠标支持（运动累积引擎、HID Report Descriptor 解析器、Boot/Report 协议自适应）
- LED 状态扩展至 10 种
- 任务调度优化
