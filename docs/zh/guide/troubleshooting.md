# 常见问题

常见问题及解决方法。

## LED 显示红色常亮

**可能的原因：**
- 未连接 USB 设备
- USB 设备不兼容
- USB 通信错误

**尝试：**
1. 检查键盘/鼠标是否已牢固连接
2. 换一个 USB 设备试试
3. 重新上电（断开后重新连接电源）

## 蓝牙中搜索不到 "Loommii-HID-01"

**可能的原因：**
- 开发板未处于广播模式
- 开发板已连接到其他主机
- 蓝牙信号干扰

**尝试：**
1. 等待 LED 从红色常亮变为闪烁模式
2. 如果 LED 已是绿色/蓝色/黄色常亮，说明已连接 — 先断开已配对的主机
3. 重新上电强制进入广播模式
4. 靠近开发板

## 配对码不显示

部分主机可能需要几秒钟才能生成配对码。如果一直没有显示：

1. 确认选择了正确的设备名称
2. 取消配对后重试
3. 换一台主机（如手机）试试，以排除问题

## 配对后键盘输入无反应

**尝试：**
1. 取消配对后重新配对
2. 检查 LED 状态 — 是否显示已连接状态？
3. 使用 `Scroll Lock + 1` 切换到其他设备槽再切回来

## 鼠标移动卡顿或延迟

- 确保开发板和主机之间没有遮挡
- 缩短开发板与主机之间的距离
- 检查是否有其他蓝牙设备干扰

## 连接频繁断开

- 尝试将开发板靠近主机
- 检查 USB 供电是否稳定 — 供电不足会导致间歇性故障
- 尝试切换设备槽再切回来（固件自动恢复会尝试重连）

## Web 烧录提示 "获取清单失败"

- 请使用 **Chrome 或 Edge** 浏览器（Firefox 和 Safari 不支持 Web Serial）
- 确保通过 **HTTPS**（或 localhost）访问页面
- 换一根 USB 数据线 — 有些线只支持充电，不能传输数据

## 仍有问题？

如果以上步骤无法解决，请在 GitHub 上[提交 Issue](https://github.com/loommii/ESP32S3-USB-HID-To-BLE-Releases/issues)。
