# 配对与切换

该桥接器支持同时配对最多 3 台蓝牙主机，并可在它们之间快速切换。

## 设备名称

每个设备槽有独立的 BLE 名称：

| 设备槽 | BLE 名称 |
|--------|----------|
| 1 | **Loommii-HID-01** |
| 2 | **Loommii-HID-02** |
| 3 | **Loommii-HID-03** |

每个设备槽使用**独立的 MAC 地址** — 主机将它们视为不同的物理设备。这意味着三个设备槽可以同时配对而不会冲突。

## 配对新设备

1. 确保开发板已上电并连接了 USB 设备
2. USB 初始化完成后，开发板会自动开始 BLE 广播
3. 在目标主机上搜索对应的设备名称（如 "Loommii-HID-01"）
4. 选择该设备 — 主机会显示一个 **6 位配对码**
5. **在 USB 键盘上输入这 6 位配对码**，然后按 **Enter** 提交
6. LED 指示灯确认配对成功 — 即可开始使用

> 配对码输入过程中，随时按键盘上的 **Esc** 可取消配对。

## 切换主机

使用 `Scroll Lock` 键作为修饰键：

| 快捷键 | 功能 |
|--------|------|
| `Scroll Lock + 1` | 切换到设备槽 1 |
| `Scroll Lock + 2` | 切换到设备槽 2 |
| `Scroll Lock + 3` | 切换到设备槽 3 |

开发板将重启并重新连接到选中的设备槽。**持久化绑定**功能确保自动重连 — 首次配对后无需再次配对。

## 解绑

如需解绑当前活动设备：

| 快捷键 | 功能 |
|--------|------|
| `Scroll Lock + Esc` | 解绑当前设备（清除配对信息） |

此操作会清除当前设备槽的绑定信息。如需再次使用该主机，需要重新配对。

## 配对过程中的 LED 指示

| 状态 | LED |
|------|-----|
| 广播中（可配对） | 紫色/蓝色/黄色闪烁 |
| 配对码输入中 | 紫色常亮 |
| 已连接就绪 | 绿色/蓝色/黄色常亮 |

完整的 LED 状态表请参见 [LED 状态参考](/zh/guide/led-status)。
