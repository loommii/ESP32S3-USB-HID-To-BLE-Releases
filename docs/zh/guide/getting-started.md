# 首次使用

请按照以下步骤首次设置您的 USB HID 转 BLE 桥接器。

## 准备工作

- 已烧录固件的 ESP32-S3 开发板
- USB 键盘和/或鼠标
- 支持蓝牙的电脑、手机或平板

## 详细步骤

### 1. 上电

将 USB 键盘和/或鼠标插入 ESP32-S3 开发板，然后连接电源。指示灯将显示**红色常亮** — 表示 USB 正在初始化。

### 2. BLE 广播

USB 设备就绪后，指示灯会根据连接的设备切换为闪烁模式：

| 连接设备 | LED 颜色 |
|---------|----------|
| 仅键盘 | 紫色闪烁 |
| 仅鼠标 | 蓝色闪烁 |
| 键盘 + 鼠标 | 黄色闪烁 |

此时开发板正在以蓝牙设备身份广播。

### 3. 配对

在电脑或手机上搜索可用蓝牙设备，选择 **"Loommii-HID-01"**。主机将显示一个 **6 位配对码**。

### 4. 输入配对码

**在 USB 键盘上输入这 6 位配对码**，然后按 **Enter** 提交。指示灯将变为**紫色常亮**，表示正在等待输入。

> 配对码输入过程中随时按 **Esc** 可取消配对。

### 5. 开始使用

配对成功后，指示灯变为常亮颜色，确认连接状态：

| 连接设备 | LED 颜色 |
|---------|----------|
| 仅键盘 | 绿色常亮 |
| 仅鼠标 | 蓝色常亮 |
| 键盘 + 鼠标 | 黄色常亮 |

恭喜！您的有线 USB 设备现在已是无线设备。

### 6. 配对更多主机

如需配对第二或第三台主机：

1. 按 `Scroll Lock + 2`（或 `Scroll Lock + 3`）切换到下一个设备槽
2. 在目标设备上搜索 **"Loommii-HID-02"**（或 **"-03"**）
3. 重复上述配对步骤 3-5

每个设备槽使用独立的 MAC 地址 — 主机将它们视为不同的物理设备。

## 快捷键参考

| 快捷键 | 功能 |
|--------|------|
| `Scroll Lock + 1` | 切换到设备 1 |
| `Scroll Lock + 2` | 切换到设备 2 |
| `Scroll Lock + 3` | 切换到设备 3 |
| `Scroll Lock + Esc` | 解绑当前设备（清除配对信息） |
