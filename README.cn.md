# USB HID → BLE 蓝牙桥接器

**版本: V2.0.1**

> **V2.0.1** — 提升连接稳定性：USB 传输中断自动恢复，设备无响应时看门狗自动重启。
>
> **V2.0.0** — 从 [ESP32S3-USB-Keyboard-To-BLE](https://github.com/loommii/ESP32S3-USB-Keyboard-To-BLE) 闭源分支，升级为键鼠组合 HID 桥接。
>
> — 新增 USB 鼠标支持（含运动累积引擎、HID Report Descriptor 解析器、Boot/Report 协议自适应），LED 状态扩展至 10 种，任务调度优化。

<p align="center"><img src="logo.png" width="260" alt="logo"></p>

将 USB 有线键盘和鼠标通过 ESP32-S3 转换为蓝牙（BLE）无线键鼠组合设备，支持最多 3 台设备切换。

## 首次使用流程

1. 插入 USB 键盘和/或鼠标 → 上电 → **红灯**亮（USB 初始化）
2. 灯变 **紫色闪烁**（仅键盘）/ **蓝色闪烁**（仅鼠标）/ **黄色闪烁**（键鼠双连）→ 板子开始 BLE 广播
3. 在电脑/手机蓝牙中搜索 **"Loommii-HID-01"** 并配对
4. 主机屏幕会显示一个 **6 位配对码**
5. **在 USB 键盘上输入该配对码**，然后按 **Enter** 提交
6. 灯变 **绿色**（仅键盘）/ **蓝色常亮**（仅鼠标）/ **黄色常亮**（键鼠双连）→ 配对成功，开始使用
7. 如需切换设备：按 `Scroll Lock + 2`（或 `Scroll Lock + 3`）切换到第二/三台设备，在该设备上搜索 **"Loommii-HID-02"**（或 **"-03"**）并重复第 3–6 步

> 配对码输入过程中随时按 **Esc** 可取消配对。

## 功能特性

- **键鼠组合桥接** — 同时支持 USB 键盘和鼠标，BLE 端呈现为单一组合 HID 设备
- **鼠标运动平滑引擎** — 环形缓冲区 + 定时器重采样 + 残差补偿，确保鼠标移动流畅无卡顿
- **跳过非输入接口** — 自动忽略游戏外设的宏键/RGB 控制等 NONE 协议接口，节省 USB 通道资源
- **Boot/Report 协议自适应** — 自动切换两种 USB HID 协议，兼容主流和游戏外设
- **三设备切换** — 同时绑定最多 3 台主机，按快捷键即时切换
- **独立 MAC 地址** — 每台设备使用独立 MAC 地址，操作系统视为不同物理设备
- **绑定持久化** — 切换后自动重连已配对设备，无需重新配对
- **CapsLock / NumLock / ScrollLock 灯同步** — 蓝牙主机关闭 NumLock，键盘指示灯同步熄灭
- **键盘键入配对码** — 配对码直接在 USB 键盘上输入，无需屏幕或其他按键
- **快捷键切换/解绑** — Scroll Lock 修饰键组合操作，无需额外按键

## LED 状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| USB 未连接 | 红色常亮 | USB 设备未插入或未就绪 |
| 仅键盘，等待蓝牙 | 紫色闪烁 | USB 键盘已就绪，BLE 广播中 |
| 仅鼠标，等待蓝牙 | 蓝色闪烁 | USB 鼠标已就绪，BLE 广播中 |
| 键鼠双连，等待蓝牙 | 黄色闪烁 | USB 键鼠均已就绪，BLE 广播中 |
| 键盘 + BLE 已连接 | 绿色常亮 | 正常工作中 |
| 鼠标 + BLE 已连接 | 蓝色常亮 | 仅鼠标连接模式 |
| 键鼠 + BLE 已连接 | 黄色常亮 | 全连接模式 |
| 配对码输入中 | 紫色常亮 | 等待用户在键盘上输入配对码 |
| 切换中 | 黄色常亮 | 正在切换设备（即将重启） |
| 错误 | 红色常亮 | USB 通信异常 |

## 快捷键操作

**Scroll Lock** 作为修饰键（先按 Scroll Lock 再按功能键）：

| 快捷键 | 功能 |
|--------|------|
| `Scroll Lock + 1` | 切换到设备 Slot 1 |
| `Scroll Lock + 2` | 切换到设备 Slot 2 |
| `Scroll Lock + 3` | 切换到设备 Slot 3 |
| `Scroll Lock + Esc` | 解绑当前设备（清除配对信息） |

## 设备名称对应

| Slot | BLE 名称 |
|------|----------|
| 1 | Loommii-HID-01 |
| 2 | Loommii-HID-02 |
| 3 | Loommii-HID-03 |

## 技术规格

| 项目 | 说明 |
|------|------|
| SDK | ESP-IDF v6.0.1 |
| BLE 协议栈 | Apache NimBLE |
| BLE 外观值 | Generic（组合设备，确保 macOS 完整识别） |
| BLE HID 报告 | 键盘 ID=1、Consumer Control ID=2、鼠标 ID=3 |
| USB HID 协议 | Boot Protocol + Report Protocol（自适应） |
| 鼠标采样率 | 环形缓冲区 128 事件，约 133Hz 定时器重采样 |
| 蓝牙版本 | BLE 5.0（兼容 BLE 4.x 以上设备） |
| 供电方式 | 通过 ESP32-S3 开发板的 USB 口供电（无需额外电源） |
| 指示灯 | 板载 WS2812 RGB LED，10 种状态指示工作状态 |
